# AuSuivant — Discovery Questionnaire

A mobile-first French-language questionnaire for the Phase 1 discovery of
AuSuivant, a walk-in clinic queue management product for French
médecins libéraux.

## What this is

A single-file HTML questionnaire with two role-gated paths:

- Assistantes médicales — 8 questions on front-desk workflow
- Docteurs — 5 questions on interruptions, notification preferences,
  willingness-to-pay, and the "Je m'absente" feature validation

Responses are posted to a Google Sheet via Google Apps Script.
